indx web journal

This plugin keeps records your web activity to your indx so you have an unlimited history of your web - you never need to bookmark anything again!

won't capture incognito mode pages - If you go into Incognito ("private browsing") mode of Chrome, this plugin will be disabled and nothing will be captured.
